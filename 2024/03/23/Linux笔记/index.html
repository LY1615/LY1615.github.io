发表于 2024-03-23 更新于 2024-04-21 分类于 Linux

常用知识

查看本机IP：ifconfig；
远程连接：ssh root@192.168.81.100(本机地址)；
进程 ID、占用 CPU 的百分比、进程的状态、运行时间：ps aux；
杀进程：kill xxx；
查找特定进程：ps -e | grep ssh；
开了shell与不开shell的区别就在于，环境变量的继承关系

vim编辑器

  • i – 切换到输入模式，在光标当前位置开始输入文本。
  • x – 删除当前光标所在处的字符。
  • : – 切换到底线命令模式，以在最底一行输入命令。
  • a – 进入插入模式，在光标下一个位置开始输入文本。
  • o：在当前行的下方插入一个新行，并进入插入模式。
  • O – 在当前行的上方插入一个新行，并进入插入模式。
  • dd – 剪切当前行。
  • yy – 复制当前行。
  • p（小写） – 粘贴剪贴板内容到光标下方。
  • P（大写）– 粘贴剪贴板内容到光标上方。
  • u – 撤销上一次操作。
  • Ctrl + r – 重做上一次撤销的操作。
  • :w – 保存文件。
  • :q – 退出 Vim 编辑器。
  • *q! – 强制退出Vim 编辑器，不保存修改。

在命令模式下按下 i 就进入了输入模式，使用 Esc 键可以返回到普通模式。

  • :w：保存文件。

  • :q：退出 Vim 编辑器。

  • :wq：保存文件并退出 Vim 编辑器。

  • :q!：强制退出Vim编辑器，不保存修改。

防火墙

查看状态：systemctl status firewalld
关闭：systemctl stop firewalld
关闭防火墙开机自启动：systemctl disable firewalld

常用命令

列出systemd服务：systemctl list- unit- files
关机：shutdown now
重启：reboot
停机：halt
关机，断电：poweroff

shell

执行脚本方式：sh bash ./ .

man：查询命令详情
type 判断是内置命令还是外置命令

shell变量

shell两种变量：系统变量和用户自定义变量(全局变量和局部变量)
打印系统全局变量：env
常用系统变量：$HOME，SPWD，$SHELL，$USER
打印用户自定义变量：set
将局部变量升级成一个全局变量：export
用前两种输出方式，会打开一个子shell来执行脚本，此时是在子shell中
echo：字符串的输出
在shell中使用运算符 a=$((1+5))
定义常量：readonly b=5S
撤销变量：unset a
grep：过滤

特殊变量

添加执行脚本权限：

打印脚本名称：$n
$1：行参1
$1：行参
获取所有输入参数的个数：$#
$ *：这个变量代表命令行中所有的参数，$ *把所有的参数看成一个整体(字符串)
$ @：这个变量代表命令行中所有的参数，$ @把所有的参数区分对待(数组)
$?：最后一次执行命令的返回状态，如果是0，证明上一个命令正确执行，如果不是0，证明上一个命令执行不正确
